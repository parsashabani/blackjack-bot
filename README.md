🃏 Blackjack Telegram Bot

A Telegram bot that lets users play Blackjack in private chats or compete with friends in group mode.

🎮 Features
	•	🃏 Play Blackjack in private chat
	•	👥 Multiplayer mode in groups
	•	🎲 Random card generation
	•	🧠 Dealer logic included
	•	⚡ Fast and lightweight

⸻

🚀 Commands

Private Chat

/deal - Start a new blackjack game
/hit - Take another card
/stand - Hold your hand

Group Chat

/newgame - Start a group blackjack game
/join - Join the current game
/startgame - Begin the round


⸻

🛠 Installation
	1.	Clone the repository:

git clone https://github.com/parsashabani/blackjack-bot.git

	2.	Install requirements:

pip install -r requirements.txt

	3.	Add your Telegram Bot Token:

TOKEN = "YOUR_BOT_TOKEN"

	4.	Run the bot:

python blackjack_bot.py
